About

How to play:
Enter one digit (0, 1, 2, 3, 4, 5, 6, 7, 8, or 9) in each of the input boxes without repeating digits to compose a guess, which is a sequence of four unique digits.

When you click guess, an evaluation of your guess is presented. The number before 'A' represents the number of right digits in the right place. The number before 'B' represents the number of right digits in the wrong place.

Your goal is to get an evaulation of '4A'.

Created by Margaret Yu. Based on a number sequence logic game.
